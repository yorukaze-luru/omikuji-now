
# 🎴 今日のおみくじ

「今日のおみくじ」は、生年月日と現在の日時をもとに、和風スタイルで本格的なおみくじを引ける**シングルHTMLアプリ**です。恋愛運・仕事運・金運・健康運など、10項目にわたる運勢を占い、紙がスルッと出てくるようなアニメーションで結果を表示します。

> 🧠 このツールは **Microsoft Copilot** を活用してAI生成されました。

---

## 🌟 特徴

- ✅ **HTMLファイル1つで完結**：インストール不要、ブラウザで開くだけ
- ✅ **和風デザイン**：Noto Serif JPフォントで落ち着いた雰囲気
- ✅ **アニメーション演出**：おみくじの紙が出てくるような演出付き
- ✅ **スマホ対応**：レスポンシブデザインでモバイルでも快適
- ✅ **10項目の運勢**：
  - 総合運
  - 恋愛運
  - 仕事運
  - 金運
  - 健康運
  - 勉強運
  - 対人運
  - 旅行運
  - 願い事
  - ラッキーアイテム

---

## 🚀 使い方

1. このリポジトリをクローンまたはZIPでダウンロードします。

```bash
git clone https://github.com/yorukaze-luru/omikuji-now.git
```

2. `index.html` をブラウザで開くだけ！

---

## 🛠️ カスタマイズ

- 運勢の種類や文言を自由に編集可能
- ラッキーカラーやラッキーナンバーの追加も簡単
- 背景画像や和風の装飾を追加して神社風にもアレンジ可能
- 結果をSNSでシェアする機能の追加も可能

---
